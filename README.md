# Superstore Sales Dashboard

**Tool:** Tableau  
**Dataset:** Sample Superstore (9,994 orders, 2014–2018)

## What's inside
- 4 KPI tiles: Sales, Profit, Orders, Avg Order Value
- Sales trend line with forecast
- Sales by Category bar chart with Top N parameter
- Profit vs Sales scatter plot with reference lines
- Region × Category heatmap
- Customer Segment pie chart

## Key Findings
- West region maintains $0.25M+ across all 3 categories simultaneously
- Loss-making sub-categories all fall below $0.10M in sales
- Technology leads revenue at $0.84M but East region drives $0.26M of that alone
